## Who We Are

**ORYVEX** is a collective of developers, security researchers, and privacy advocates united by a single mission: making the internet free, secure, and accessible for everyone.

| Belief | Why It Matters |
|--------|----------------|
| Internet is a Human Right | Access to information shouldn't be a privilege |

## Features

| Feature | Description |
|---------|-------------|
| One-Click Connect | Connect to the fastest server instantly |
| Military-Grade Encryption | All traffic encrypted with Xray-core |
| Multiple Protocols | VLESS, VMESS, Trojan, Shadowsocks, XHTTP |
| Discord Rich Presence | Show your VPN status on Discord |
| Real-Time Usage | Track your data usage in real-time |
| Auto-Reconnect | Automatically reconnects if disconnected |
| Kill Switch | Blocks internet if VPN disconnects |
| Unlimited Usage | No data caps, no time limits |

### Security Features

| Feature | Description |
|---------|-------------|
| Zero Logs | We never store your browsing history |
| Obfuscation | Bypass deep packet inspection |
| Perfect Forward Secrecy | Each session uses unique keys |
| DNS Protection | Encrypted DNS queries |
